# Basic Running Record Calculator for Reading Assessments

Created for friend who was previously using a free app whose ads made it unusable.

## Build and run locally with
```
npm run dev
```

## Calculations and assessment metrics sourced from:  [Reading A-Z.com](https://www.readinga-z.com/)
